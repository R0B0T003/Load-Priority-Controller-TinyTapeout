## How it works

The Load Priority Controller is a finite state machine (FSM) that manages power distribution across four loads based on external power status signals.

The system receives three external flags generated by a comparator circuit:

undervoltage_flag → indicates very low available power
medium_power_flag → indicates moderate power availability
high_power_flag → indicates sufficient power availability

Based on these signals, the FSM switches between power modes:

Low Power Mode → Only highest priority load (L1) is enabled
Medium Power Mode → L1 and L2 are enabled
High Power Mode → L1, L2, L3, and L4 are enabled
## How to test

Test procedure:

Apply reset (reset = 1) then release (reset = 0)
Drive clock signal continuously
Toggle power flags:
Case 1: Undervoltage
undervoltage_flag = 1
Output: load_enable = 0001
Case 2: Medium power
medium_power_flag = 1
Output: load_enable = 0011
Case 3: High power
high_power_flag = 1
Output: load_enable = 1111
Verify that outputs change only on clock edges 
Ensure only one power state is active at a time for predictable behavior
